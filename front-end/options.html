GuardSurfing Options

Backend

Must support /process and /analyze_page. Click Check to fetch /config.

Mode

Strict marks more URLs as phishing; Relaxed is more permissive.

Performance

If the full check is slow, automatically send a fast offline check and show its result first.

LLM (optional)

Uses local Ollama by default (backend). LLM may increase latency; keep off if not needed.

Page protection

When enabled, pages will outline risky links and optionally block clicks unless you confirm.

Allowlist

One domain per line (e.g., example.com). Subdomains are included.
